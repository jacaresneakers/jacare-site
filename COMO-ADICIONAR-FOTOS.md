# Como adicionar fotos ao site Jacaré Sneakers

## Estrutura de pastas

Cada linha tem sua pasta dentro de `/images/`:

- `dunk-low/` → Nike Dunk Low
- `dunk-sb/` → Nike Dunk SB
- `jordan-1-low/` → Air Jordan 1 Low
- `air-force-1/` → Air Force 1
- `yeezy-350-v2/` → Yeezy Boost 350 V2
- `nb-9060/` → New Balance 9060
- `gucci/` → Gucci
- `golden-goose/` → Golden Goose
- `lv/` → Louis Vuitton
- `dior/` → Dior
- `dg/` → Dolce & Gabbana
- `off-white/` → Off-White
- `vomero/` → Nike Vomero Premium
- `adizero/` → Adizero EVO SL
- `on-running/` → On Running

## Nome dos arquivos

Para cada par, use o padrão:
`nome-do-par-1.jpg` (foto principal)
`nome-do-par-2.jpg` (segundo ângulo)
`nome-do-par-3.jpg` (terceiro ângulo)
`nome-do-par-4.jpg` (quarto ângulo)

Exemplos:
- `panda-1.jpg`, `panda-2.jpg`, `panda-3.jpg`
- `zebra-1.jpg`, `zebra-2.jpg`
- `travis-scott-1.jpg`

## Qualidade recomendada
- Formato: JPG ou PNG
- Tamanho: mínimo 800x800px
- Fundo branco ou neutro
- Sem marca d'água
